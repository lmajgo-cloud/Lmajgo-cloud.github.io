About Us

We are two brothers from Czechia who picked up coding as a hobby. Thanks to AI, we can build games much faster than before — which matters when free time is scarce. We don't pretend to be a serious game studio; this is strictly a passion project. But whenever we craft something we find interesting or fun to play, we share it with the world so others can enjoy it too.
